# 🌍 Global Workforce Intelligence Dashboard

## 📊 Project Overview

This Power BI project analyzes the Stack Overflow Developer Survey 2025 to uncover global workforce trends, compensation patterns, career progression, workplace preferences, and demographic insights.

The dashboard enables interactive exploration of salary distribution, work modes, education levels, industry trends, job satisfaction, organizational structure, and career mobility.

---

## 📌 Objectives

- Analyze global salary trends
- Compare compensation across countries and industries
- Understand workforce demographics
- Explore remote work preferences
- Analyze career change intentions
- Identify factors affecting job satisfaction

---

## 🛠 Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX
- Data Modeling

---
